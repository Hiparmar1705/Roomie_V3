# CPSC 357 - Mobile Application Development

## Course Overview

This course introduces modern mobile application development using React Native and the Expo ecosystem. Students learn to design, build, test, and deploy cross-platform mobile applications for Android and iOS using industry-standard tools and practices. The course integrates core software engineering concepts (SDLC, SRS, SDD) with hands-on development, covering navigation, state management, backend integration, device APIs, security, testing, and app publishing.

**Capstone Project:** Students complete a React Native project developed through structured, sprint-based workflows.

## Course Topics Overview

### Weeks 1-3: Foundations
- React Native introduction and environment setup
- Basic UI components (Text, TextInput, View, Button)
- Styling and Layout with Flexbox
- ScrollView and FlatList
- SRS and SDD documentation

### Weeks 4-5: Navigation & State Management
- User input handling and Forms
- Navigation (Stack, Tab, Drawer)
- React Context and Redux
- Agile Software Development

### Weeks 6-7: Device APIs
- Camera, Media Library, Image Picker
- Permissions handling
- Media app development

### Weeks 8-9: Location & Backend Integration
- Push Notifications
- Location services and Maps
- Data Fetching & REST API Integration
- Async/await and error handling

### Weeks 10-11: Storage & Authentication
- Storage & Persistence (AsyncStorage, SQLite)
- Firebase Authentication
- Security implementation

### Weeks 12-13: Testing & Publishing
- Testing & Quality Assurance (unit, component, E2E)
- App build & publishing (Expo EAS)
- Store requirements and best practices

### Week 14: Project Review
- Code review and feedback
- Final project presentation preparation
